## Featured Projects

### [Noise Dosimetry Report System](https://github.com/glizardx/dosimetry-system)

**Full-stack platform for occupational health reporting workflows.**

Built for an occupational health company, this system replaced a manual process where each report could take up to 5 days to deliver, directly improving operational speed, traceability and cash flow.

**Impact:** 80 reports/month · 5-day cycle reduced to under 15 minutes

**Stack:** FastAPI, React, PostgreSQL, SQLAlchemy, JWT, Pytest, PDF parsing

**Key features:**

* PDF parser with multi-pattern confidence scoring
* Name similarity matching for employee identification
* Automated report generation aligned with NR-15 / NHO-01 requirements
* SHA-256 hashing and immutable audit trail
* JWT authentication with role-based access control
* PostgreSQL data model for reports, users, measurements and audit records

### [Microsoft Teams Archiving Automation](https://github.com/glizardx/teams-scraper)

**Authorized internal automation tool for preserving institutional communication.**

Created to archive Microsoft Teams messages into a searchable local database, supporting continuity, internal documentation and knowledge preservation.

**Impact:** 10,000+ messages archived across channels and group chats

**Stack:** Python, Playwright, SQLite

**Key features:**

* Browser automation with Playwright
* Structured message extraction and local persistence
* SQLite storage with deduplication
* DOM fallback for messages not captured by the main extraction flow
* Local web viewer for searching and reviewing archived content
* Scheduled execution for recurring backups

> Built for authorized internal use only.

### [Credit Card Default Analysis](https://github.com/glizardx/credit-card-analysis)

**Exploratory credit risk analysis using a dataset of 30,000 credit card clients.**

Academic project using the UCI Default of Credit Card Clients dataset to analyze default patterns, customer behavior and repayment history.

**Stack:** Python, Pandas, Matplotlib

**Key findings and analysis:**

* 22.1% default rate identified across 25 variables
* Demographic and financial pattern analysis
* Repayment behavior tracked over 6 consecutive months
* Analysis of credit limit, age, education, gender and payment history
* Dataset: UCI Machine Learning Repository · CC BY 4.0

**Next steps:**

* Add logistic regression baseline
* Compare risk indicators using feature importance
* Evaluate model performance with precision, recall and ROC-AUC
