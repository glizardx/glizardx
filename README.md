### Credit Risk Default Analysis
Credit risk analysis using 30,000 credit card client records to identify default patterns, repayment behavior and risk segments.

**Tools:** Python, Pandas, Matplotlib, SQL  
**Topics:** credit risk, default prediction, repayment behavior, risk segmentation

Highlights:
- Default rate and client profile analysis
- Repayment behavior over six months
- Risk segmentation by payment delays, credit limit and bill/payment patterns
